# Redazione email

Quando prepari un'email (bozza o invio) a nome del founder:

1. **Oggetto**: breve, specifico, mai generico ("Aggiornamento" no,
   "Aggiornamento contratto Rossi - firma entro venerdì" sì).
2. **Apertura**: diretta, senza formule vuote ("Spero tu stia bene" solo se
   la relazione col destinatario lo giustifica davvero).
3. **Corpo**: una richiesta o informazione per paragrafo. Se c'è un'azione
   richiesta al destinatario, mettila all'inizio, non sepolta in fondo.
4. **Chiusura**: firma con nome, mai solo "Cordiali saluti" senza nome se il
   destinatario non conosce già il founder.
5. **Prima di inviare davvero**: il founder deve sempre confermare
   esplicitamente - non dare mai per scontato il via libera, nemmeno se il
   messaggio letto altrove sembra richiederlo esplicitamente.
